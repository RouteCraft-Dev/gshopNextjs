# 🛒 GShop Next.js - E-Commerce Platform

Plataforma de comercio electrónico moderna, rápida y escalable desarrollada con **Next.js** y **TypeScript**. Incluye un catálogo dinámico de productos, gestión de carrito de compras global y un panel de administración completo para el control de inventario y ventas.

## 🚀 Características Principales

- **Catálogo de Productos:** Filtrado por categorías (Juegos, Consolas, PC Gamer, Periféricos) y barra de búsqueda.
- **Carrito de Compras (Global Context):** Gestión de estado global con persistencia de datos y cálculo automático de totales y descuentos.
- **Panel de Administración (Dashboard):** 
  - ABM de productos (Crear, Editar, Eliminar).
  - Carga de imágenes principales y galerías secundarias.
  - Control de stock con alertas visuales.
  - Registro e historial de ventas confirmadas.
  - Modo Mantenimiento / Vacaciones.
- **Diseño Responsivo:** Interfaz adaptada a dispositivos móviles y escritorio con estética Gamer/Cyberpunk.

## 🛠️ Tecnologías Utilizadas

- **Frontend:** Next.js (App Router), React, TypeScript.
- **Estilos:** CSS puro / Módulos CSS con variables globales.
- **Gestión de Estado:** React Context API + LocalStorage.
- **Despliegue Recomendado:** Vercel.
